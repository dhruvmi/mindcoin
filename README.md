Mindcoin 
================================

What is Mindcoin?

Mindcoin - a lite version of Mindcoin optimized for CPU mining using scrypt as a proof of work scheme.

Mindcoin properties:

-Initial reward of 20 coins

-Block interval of 5 minutes

-Maximum supply of 350k coins
